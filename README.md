# TCP/HTTP Engine using C

A lightweight, high-performance TCP and HTTP server engine written from scratch in pure C. This project uses standard POSIX sockets to handle raw network traffic and parse HTTP requests.

## 📋 Prerequisites

To build and run this engine, you need:

- A Linux or macOS environment (POSIX-compliant)
- A C compiler (`gcc` or `clang`)
- `make` (optional, for build automation)
